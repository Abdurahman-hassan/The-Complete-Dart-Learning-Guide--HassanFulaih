# 017.NumbersMethods

```
main() {
var x = -4;
print(x.isOdd);
print(x.isEven);
print(x.isFinite);
print(x.isInfinite);
print(x.isNaN);
print(x.isNegative);
print(x.abs());
var y = 8.5;
print(y.round());
print(y.floor());
print(y.truncate());
print(y.ceil());
print(y.remainder(x));
}
```


## The output is:

* false
* true
* true
* false
* false
* true
* 4
* 9
* 8

* 9
* 0.5
